## 🚀 Getting Started

### 1. First Time Setup
```bash
cd nssports
npm install
```

### 2. Configure Database
```bash
# Add to .env:
DATABASE_URL="postgresql://..."
DIRECT_URL="postgresql://..."
```

### 3. Setup Database
```bash
npm run db:generate
npm run db:push
npm run db:seed
```

### 4. Start Development
```bash
npm run dev
```

### 5. Run Tests
```bash
./test-api-workflow.sh
```
